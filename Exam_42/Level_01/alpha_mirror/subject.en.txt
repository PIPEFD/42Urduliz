Assignment name  : alpha_mirror
Expected files   : alpha_mirror.c
Allowed functions: write
--------------------------------------------------------------------------------

Escribe un programa llamado alpha_mirror que toma una cadena y muestra
esta cadena después de reemplazar cada carácter alfabético por el carácter 
alfabético opuesto, seguido de una nueva línea.

'a' se convierte en 'z', 
'Z' se convierte en 'A' 
'd' se convierte en 'w',
'M' se convierte en 'N' 
and so on.

Case is not changed.

If the number of arguments is not 1, display only a newline.

Examples:

$>./alpha_mirror "abc"
zyx
$>./alpha_mirror "My horse is Amazing." | cat -e
Nb slihv rh Znzarmt.$
$>./alpha_mirror | cat -e
$
$>
